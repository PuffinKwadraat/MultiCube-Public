    Intro

    Istalling add-ons on Minecraft can be hard somethimes. Thats when this step-by-step guide comes in. We will help you installing everything you need for a better or different Minecraft experience.


    First things first: before you want to install Shaders, you need OptiFine. OptiFine is a Minecraft optimization mod. It allows Minecraft to run faster and look better with full support for HD textures and many configuration options. When you have installed OptiFine, you get access to Shaders. But we aware, you have to install Forge first. Minecraft Forge is a free, open-source server that allows players to install and run Minecraft mods. It was designed with the intent to simplify compatibility between community-created game mods for Minecraft: Java Edition. This guide shows you how to install Shaders, Mods and Texture Packs.

    1.1. Downloading Forge

    Go to the official site of Forge. There, you have a list of all the versions. This gets constanly updated. Then, select the version you want. There, you will see 6 icons. Click on the 'installer'. You will be linked to another page.

    Press 'skip' at the right-top corner. After this, Forge is now downloading. When done downloading, it can be found in your Downloads directory

    1.2. Installing Forge

    Search the file you downloaded and right-click on 'open with'. Then select 'Java(TM) Platform SE library'. Then this tab should open:



    Then select 'install client'.
    It will now install Forge.

    1.3. Running Forge

    After installing, open Minecraft Launcher. Click on installations --> +New
    It should look like this:



    Then click on 'version' and then search your Forge version. Then select that version and launch the game. You now have succesfully installed Minecraft Forge. At the main menu you can also see the option 'mods'.

    2.1. Downloading OptiFine

    After you installed Forge, its time for OptiFine. First head over to their official site and then to downloads.



    Now, select the version you want to install. It has to be the same version of your installed Forge!

    Click on it, wait 5 seconds and your download will begin.

    2.2. Installing OptiFine

    For Forge, OptiFine is a mod. So the next step is finding your mod directory. First, press your Windows Home-key an typ 'run' and open the app.



    Then, type %AppData% and press Enter. Here you can find your Minecraft home directory.



    You should be in the Explorer App with a list of your apps, including Minecraft. For the sake of this guide, only Minecraft is displayed. It should look like this:



    Then, open the '.minecraft' folder. You are now in the Minecraft home directory. It should look like this:



    Only thing left to do for installing OptiFine is grabbing your OptiFine from your download directory and putting it in the 'mod' folder. You can do this by opening a new Explorer tab, or by cut and paste. This also works the same way with other mods you may install in the future. In the image down below you can see an example of how it should look:

    2.3. Running OptiFine

    So, if everything went well, Forge and OptiFine are both all set. The differences between Vanilla and OptiFine are mostly the settings. If you go to the settings --> video, you will see a lot more customization. Also, you can add Shaders. How to install Shaders can be found in the next paragraph.

    3. Shaders and Texture Packs

    So now we have covered Mods, but how about Shaders, or Texture Packs? Well, it's really almost the same. Instead of using the 'mod' folder, you have to use the 'shaderpacks' and 'resourcepack' folders. When downloading Shaders or Textures, do not unpack the file. Just drag and drop them in folder.     And there you have it, a full guide of how to install add-ons. We hope it was clear, and that it helped you enchanting Minecraft.
